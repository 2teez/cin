# cin

## Name
cin - works like cin in c++ std::cin to get "any" value from the cli.

### Installation

#### Use rust cargo

  ```
   cargo run cin
  ```

  #### Using from Github link:
 In the Cargo.toml file
 ```
  cin = {git = "https://github.com/2teez/cin"}
 ```

 use in your rust file like so
 ```
  use cin::cin::cin;
 ```
